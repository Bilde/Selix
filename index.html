Selix is a lightweight and easy-to-use JavaScript library for manipulating the selection inside of user input elements

Selix helps you manipulate the selection inside of user inputs such as textareas and inputs and it is therefore very useful in cases such as text editors

You can install it using Bower

{% highlight bash %} bower install selix {% endhighlight %}
